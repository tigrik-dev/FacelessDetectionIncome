Faceless Detection Power is added to the Haven Management window, in a similar manner to Mission Find Power, etc.
When a soldier is assigned as a Haven Liaison, this value represents how proficient this soldier is at detecting Faceless.

+ Why? +

Suppose you want to assign a soldier as a Haven Liaison who is the best at detecting Faceless (i.e. at finding Rendezvous missions) from currently available soldiers.
How would you find out who that would be?

Sometimes it’s obvious, knowing that higher rank soldiers, Officers, and Psi Operatives are better at it.

Sometimes it’s not so clear. For example, who is more proficient at detecting Faceless from these soldiers?

* Master Sergeant
* Tech Sergeant Specialist with a Scanning Protocol
* Mystic Psi Operative
* Tech Sergeant Second Lieutenant
* Adept Psi Operative Captain

Now you could simply check who has the highest Faceless Detection Power by assigning each one in turn.

+ Note +

Starting from Long War of the Chosen version 1.2.3, items that reduce the chance of recruiting Faceless rebels now also provide a 10% bonus to detecting Rendezvous missions.

+ Requirements +

* Long War of the Chosen version 1.2 or newer.
* Should be safe to add or remove mid-campaign.

+ Localization +
* Russian localization is included.

+ Known Issues +

* Faceless Detection Power is not getting updated the moment Haven Liaison has been unassigned - only when the new one is assigned or the window is re-opened.
